# 💳 Debiting API

## 🌱 About

Debiting feature to work on simulated banking system using **Java / Spring Boot**

## 🖥️ Tech stack

- **Back-End** : Java 21, Spring (Spring Boot, Spring MVC, Spring Data JPA), API REST
- **Database** : H2
- **DevOps / DevTools** : Git / GitHub, Maven, Linux (Fedora)
- **Testing** : JUnit, AssertJ
